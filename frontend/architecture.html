How the boundary holds.

Phylax splits the world into partner spaces and one neutral control plane. Only opaque commitments, permitted aggregates, and hashes ever cross.

SecretFlow feature map

Privacy-preserving computation.

ECDH PSI
Private set intersection — ECDH_PSI_3PC, secretflow 1.10.0b1. Coordinator sees only the intersection.
Secure aggregation
Additive secret sharing over Mersenne-prime field.
Federated risk
phylax-risk 1.0.0 over distinct_parties, matched_signatures, total_events, mean_velocity_z, temporal_alignment.
Local backend
2048-bit MODP DDH-PSI (RFC 3526).

Demo vs production

Demo · today functional demo
  • Synthetic data — no real customers.
  • Single InsForge account.
  • Proves protocol + product.
  • Not adversarial isolation.
Production · design independent parties
  • Own InsForge project per org.
  • Threshold OPRF key.
  • Per-org Ed25519 signing.
  • Zero raw records invariant holds.
